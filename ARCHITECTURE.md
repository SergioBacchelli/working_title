# ARCHITECTURE — общая архитектура проекта

## 1. Слои проекта

- **UI (client)** — взаимодействие с пользователем, формы, экраны.
- **Application / Core** — бизнес-логика, правила, сценарии.
- **Infrastructure (server)** — работа с HTTP, БД, файловой системой, внешними сервисами.
- **Storage** — локальное/постоянное хранение данных (файлы, БД, локальное хранилище браузера и т.п.).

(По мере уточнения стека мы привяжем конкретные технологии к этим слоям.)
