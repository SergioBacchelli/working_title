# BXF2 — спецификация (рабочая)

Эта папка — “словарь” и набор правил для **генерации корректного BXF2**.

## Файлы

- `BXF2_RULES.md` — правила панелей, координат и трансформаций (по примеру `Novyj-korpus.bxf2`).

## Что дальше нужно для отверстий/пазов

В текущем примере **нет обработок** (`machiningGroups` и `machinings` пустые), поэтому следующий шаг:
- добавить второй BXF2, где есть хотя бы:
  - 1–2 сверления (сквозное и/или глухое),
  - 1 паз,
и по нему зафиксировать:
- где элементы лежат в XML,
- как привязываются к панели,
- в каких координатах задаются (локаль панели vs глобаль),
- как задаётся ориентация, глубина и тип операции.
