# PROJECT_MAP — карта проекта

## 1. Корень репозитория

- `README.md` — общее описание проекта и цели.
- `START_HERE.md` — точка входа: порядок чтения документов для инициализации контекста.
- `PROJECT_MAP.md` — карта файлов и папок (этот документ).
- `ARCHITECTURE.md` — описание архитектуры проекта.
- `DECISIONS.md` — журнал важных решений.
- `WORKFLOW.md` — правила совместной работы Сергея и Рэя.
- `docs/` — дополнительная документация (спеки/контракты/шаблоны).
- `src/` — исходный код проекта.
- `scripts/` — вспомогательные скрипты (генерация, утилиты и т.п.).
- `examples/` — эталонные входные/выходные примеры.
- `examples/portal_json/` — эталонные JSON-примеры для импорта в БАЗИС (панели/отверстия).

## 2. Папка `docs/`

- `docs/specs/` — спецификации форматов и контрактов (например, импорт в БАЗИС через Bazis Script).
- `docs/ui/` — UI-спеки модулей: контракт каждого экрана, шаблон и справочник типов полей.
  - `docs/ui/UI_TEMPLATE.md` — шаблон UI-спеки.
  - `docs/ui/UI_FIELD_FORMAT.md` — правила описания полей (select/radio/readonly и т.п.).
  - `docs/ui/modules/` — UI-спеки конкретных модулей (по одному файлу на модуль).

## 3. Папка `src/`

- `src/client/` — фронтенд (UI, визуализация, ввод параметров).
- `src/server/` — бэкенд (API, хранение, генерация входных данных для импорта в БАЗИС).
